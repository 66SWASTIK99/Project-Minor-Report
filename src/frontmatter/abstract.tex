\bigskip

A Brain Computer Interface (BCI) lets a person control a computer or device directly using brain signals which are recorded from the scalp using EEG electrodes. In this project, a low cost EEG-based BCI typing system is designed using eye blinks and alpha brain waves as control signals unlike complex systems that use visual stimuli or machine learning, this system focuses on simplicity, affordability and ease of use.EEG signals are collected using few electrodes and processed with basic signal processing techniques. Eye blinks create large changes in the EEG signal and are used for selecting or confirming actions like alpha waves which occur in the 8–12 Hz range. Waves generated from the brain through electrodes are analyzed using Fast Fourier Transform (FFT) to detect if the user is relaxed or focused. A simple stimulus based method can map these signals to typing and selection of commands  for interfacing.The goal of this project is to create a working BCI typing system while learning how to record, filter and extract features from EEG signals as well as integrate them into a usable system. BCI can be helpful for people with movement difficulties and for situations where hands free control is preferred.


\vspace{1em}
\noindent\textit{\textbf{Keywords}: BCI, Electroencephalogram (EEG), Fast Fourier Transform (FFT), Signal Processing, Typing Interface}
